* init.sh 用于新系统初始优化 </br>
    * 更新系统(生产环境请自行注释有关yum update -y -q代码,可能因会系统版本和软件升级导致异常)</br>
    * 安装常用工具包</br>
    * 增加vim插件和个性化vim配置
* vim.sh </br>
    * 用于增加vim插件和个性化vim配置</br>
    * 适用于vim7.4

* install-zsh.sh </br>
    * 用于安装zsh环境,包括:autojump、zsh语法高亮、终端配色更改为256色等


#### 本脚本仅在CentOS{6.9,7.3}和Ubuntu{14.04,16.04}上测试通过，不保证能在其它系统上能成功运行.
